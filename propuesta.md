# Propuesta

- **Título del taller:** "Taller de git"
- **Descripción:** charla introductoria a git apuntada a gente que no conoce sistemas de control de versiones.
- *8Objetivos de aprendizaje:**


- **Duración estimada:** entre 60 y 90 minutos, dependiendo de los inconvenientes que encuentren los participantes al introducir los comandos.
- **Nivel del taller:** Principiante
- **Materiales requeridos:**
  - proyector para mostrar slides y la pantalla de mi shell ejecutando comandos
  - computadoras para los participantes con alguna distribución de GNU/Linux
  - internet para conectar con GitHub
  - computadoras de los participantes idealmente con `git` preinstalado
  - si los participantes pudiesen tener ya creadas sus cuentas de GitHub, sería genial
- **Requisitos previos para asistentes:**
  - conocimientos básicos de shell: saber abrir una terminal, listar un directorio, moverse entre directorios, editar un archivo
