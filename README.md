# Taller de git

Una charla-taller sobre el uso de Git para los asistentes a la Global Game Jam de Mendoza, Argentina, enero de 2026.

Los slides se crean con [Marp]( https://github.com/marp-team/marp-cli) y no están incluidos en el repo.

**Nota:** la charla-taller se convirtió en charla-intro y será sólo una charla introductoria donde no habrá parte de práctica por parte del público.

Resultó que al tomarme el tiempo, la charla-taller tomaba 1:45 horas y mi tiempo disponible e inamovible en la GGJ 2026 es de 40 minutos.
